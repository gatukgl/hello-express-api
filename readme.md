# Hello Express API

This application aims to learn about using Express.js to create APIs

## Setting up local development

### Prerequisite

- Node.js
- NPM or Yarn

### Running the app

1. Install all dependencies by using

   ```zsh
   $ npm install
   
   // or
   
   $ yarn
   ```

2. Start application by using

   ```zsh
   $ npm start
   
   // or
   
   $ yarn start
   ```
